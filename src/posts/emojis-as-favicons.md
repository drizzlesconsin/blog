# 使用 Emoji 作为浏览器收藏夹图标

## Usage

```html
<svg xmlns="http://w3.org/2000/svg" viewBox="0 0 100 100">
    <text y=".9em" font-size="80">😎</text>
</svg>
```

## Compatibility

Firefox 是最先支持该图标的浏览器，Chrome 80+ 开始支持。

[caniuse#icon-svg](https://caniuse.com/#feat=link-icon-svg)
